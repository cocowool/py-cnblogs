Docker存出载入镜像

镜像的存出和载入

如果你的生产环境不能连通互联网，而你又希望从互联网上获取镜像。你就需要借助 docker save命令，可以将镜像导出为 tar 文件。使用 docker load 命令，可以将tar文件导入镜像库。整个流程是这样的:

存出镜像

先用 docker images 查看本地的镜像，然后使用 docker save 可以将镜像保存为tar文件。
